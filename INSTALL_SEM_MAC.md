# Installation Guide for Dolphin-IPA Emulator without Mac

## Requirements
- **Device**: iPhone 11 or higher
- **iOS Version**: 13.0 or higher
- **AltStore**: Make sure you have AltStore installed on your device.

## Steps to Install Dolphin-IPA Emulator

### 1. Add the Dolphin Repository
1. Open the AltStore app on your iPhone.
2. Go to the "Settings" section.
3. Tap on "Sources" and then "Add Source".
4. Enter the following URL for the Dolphin repository: `https://dolphin-emu.org/altstore`
5. Tap "Done" to add the repository.

### 2. Install Dolphin-IPA Emulator
1. After adding the repository, go back to the AltStore main screen.
2. Tap on "Browse" to see the list of available apps.
3. Search for "Dolphin-IPA" in the list.
4. Tap on the app and then select "Install" to start the installation process.

### 3. Set Emulation Speed
1. Once the Dolphin-IPA Emulator is installed, open the app.
2. Go to the settings menu within the app.
3. Look for the "Emulation Speed" options.
4. You can select from the following speeds:
   - **1x** - Normal speed
   - **2x** - Double speed
   - **3x** - Triple speed
   - **4x** - Quadruple speed
5. Choose your preferred emulation speed according to your device performance and preference.

## Final Notes
- Ensure your device is compatible and meets the requirements.
- For the best performance, keep your iOS updated and regularly check for updates in the AltStore app.

Enjoy using Dolphin-IPA!
